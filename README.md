# Project Body Tracking Vtuber

Project ini tentang gimana ngetracking Sebuah Karakter(Avatar) dengan menggunakan Opencv dan mediapipe, dan visualisasinya menggunakan Unity. Cara kerjanya yaitu dari Python ke avatar di Unity menggunakan UDP networking. Sistem ini memungkinkan untuk menggerakkan ekspresi wajah dan tubuh avatar secara real-time dengan input dari webcam.
